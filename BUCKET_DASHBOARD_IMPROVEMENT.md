# 버킷리스트 & 대시보드 활용도 개선 방안

## 1. 현재 기능 분석

### 버킷리스트 탭 현재 기능
- 계층 구조 지원 (하위 항목)
- 완료/미완료 상태 토글
- 순서 이동 (화살표)
- 진행률 표시
- 목표 기한 설정

### 대시보드 현재 기능
- **히트맵** (YearlyContributionGraph) ✅ 활용도 높음
- 레벨 시스템 (완료 개수 기반 XP)
- 성취/배지 시스템 (8개)
- 동기부여 명언 (랜덤)
- 일간/주간/월간 통계
- 메모 기능
- 테마 설정

---

## 2. 문제점 분석

### 버킷리스트 문제점

| 문제 | 설명 |
|------|------|
| **계획 탭과 역할 중복** | 둘 다 "하고 싶은 것" 관리, 차별점 불명확 |
| **장기 목표 관리 부족** | 버킷리스트는 원래 "죽기 전에 하고 싶은 것"인데 단순 체크리스트화 |
| **동기부여 요소 부재** | 완료해도 특별한 보상/피드백 없음 |
| **시각화 부족** | 진행 상황을 한눈에 파악하기 어려움 |
| **카테고리 없음** | 여행, 자기계발, 경험 등 분류 불가 |

### 대시보드 문제점 (히트맵 제외)

| 문제 | 설명 |
|------|------|
| **레벨 시스템 느린 진행** | 810개 완료해야 최고 레벨 → 동기부여 감소 |
| **성취 달성 확인 어려움** | 언제 달성했는지, 진행률 표시 부족 |
| **명언 정적임** | 랜덤 표시만, 사용자 상태와 무관 |
| **통계 활용도 낮음** | 보기만 할 뿐 인사이트 제공 없음 |
| **메모 기능 위치 어색** | 대시보드에 있을 필요가 있는지 의문 |

## 3. 개선 방안

### 옵션 A: 버킷리스트 강화 (장기 목표 특화)

#### 핵심 아이디어: "계획"과 차별화하여 장기 꿈/목표 관리에 집중

1. **카테고리 시스템**
   ```
   🌍 여행 | 📚 자기계발 | 💪 건강 | 🎨 취미 | 💼 커리어 | ❤️ 관계
   ```

2. **마일스톤 기능**
   - 큰 버킷 아이템을 작은 단계로 분해
   - 예: "유럽 여행" → "여권 갱신" → "항공권 예약" → "숙소 예약"

3. **비전 보드**
   - 버킷리스트를 이미지/사진으로 시각화
   - 완료 시 갤러리에 추가

4. **예상 비용/시간 추적**
   - 각 항목에 예상 비용, 소요 시간 입력
   - 총 투자 필요량 계산

5. **완료 기념 기능**
   - 완료 시 날짜, 사진, 감상 기록
   - "추억 갤러리"로 보관

---

### 옵션 B: 버킷리스트 제거 + 계획 탭 강화

#### 핵심 아이디어: 기능 중복 해소, 계획 탭에서 장기/단기 모두 관리

1. **계획 탭에 "장기 목표" 필터 추가**
   - 마감일 없음 = 장기 목표로 분류
   - 또는 "버킷" 태그 추가

2. **하단 네비게이션 단순화**
   - 버킷 제거 → 4개 탭으로 정리
   - 또는 버킷을 "더보기" 메뉴로 이동

---

### 옵션 C: 대시보드 게이미피케이션 강화

#### 핵심 아이디어: 통계를 보는 것에서 → 동기부여 받는 것으로

1. **일일 목표 시스템**
   ```
   오늘 목표: 5개 완료
   현재: 3/5 (60%)
   [■■■□□] 2개 남음!
   ```

2. **스트릭 강조**
   ```
   🔥 현재 연속: 7일
   📈 최장 기록: 14일
   ⚠️ 오늘 아직 0개 완료!
   ```

3. **레벨 시스템 리밸런싱**
   | 현재 | 개선안 |
   |------|--------|
   | Lv10에 810개 필요 | Lv10에 100개로 축소 |
   | 레벨업 느낌 없음 | 레벨업 시 축하 애니메이션 |

4. **성취 시스템 개선**
   - 진행률 표시: "50개 완료 (80/100)"
   - 달성 가능한 다음 성취 하이라이트
   - 희귀도별 시각적 차별화

5. **맞춤형 인사이트**
   ```
   📊 이번 주 분석:
   - 가장 생산적인 요일: 화요일
   - 평균 완료 시간: 오후 3시
   - 지난주 대비: +15% 향상!
   ```

6. **동기부여 메시지 개인화**
   - 시간대별: 아침/점심/저녁 다른 메시지
   - 완료율별: 0% → 격려, 100% → 축하
   - 스트릭별: 연속 기록 언급

## 4. 추천 구현 순서

### Phase 1: 대시보드 즉시 개선 (1주)
1. ✅ 히트맵 유지
2. 일일 목표 설정 & 진행률 표시
3. 스트릭 크게 강조
4. 레벨 시스템 리밸런싱 (XP 요구량 1/8로 축소)
5. 레벨업 축하 애니메이션

### Phase 2: 성취 시스템 개선 (1주)
1. 모든 성취에 진행률 표시
2. "다음 달성 가능" 성취 하이라이트
3. 달성 시 팝업 알림

### Phase 3: 버킷리스트 결정 (1주)
1. 사용자 피드백 수집
2. 옵션 A or B 선택
3. 구현

## 5. 빠른 개선 (오늘 가능)

### 대시보드 즉시 적용 가능한 개선

1. **레벨업 XP 요구량 축소**
   ```typescript
   // 현재: 0, 10, 40, 90, 160, 250, 360, 490, 640, 810
   // 개선: 0, 5, 15, 30, 50, 75, 100, 130, 165, 200
   ```

2. **스트릭 섹션 추가**
   - 히트맵 위에 "🔥 7일 연속!" 배지 표시

3. **통계 카드 단순화**
   - 일간/주간/월간 선택 → 주간만 표시
   - 핵심 수치만 강조

### 버킷리스트 즉시 적용 가능한 개선

1. **카테고리 추가**
   - DB에 category 컬럼 추가
   - 필터 UI 추가

2. **완료 시 축하 효과**
   - 계획 탭처럼 confetti 효과

## 6. 결론 및 권장사항

### 단기 (이번 주)
- 대시보드: 레벨 리밸런싱 + 스트릭 강조
- 버킷: 완료 시 축하 효과 추가

### 중기 (1개월)
- 대시보드: 일일 목표 시스템 + 맞춤 인사이트
- 버킷: 카테고리 시스템 or 계획 탭과 통합 결정

### 장기 (3개월)
- 대시보드: 게이미피케이션 완성
- 버킷: 비전 보드 or 완전 제거
